Running A Screen
Once the desired parameters have been entered with the desired data to return click on the run button located above the upper left of the screen parameters containing display (below the SCREEN BUILDER page title)

Saving A Screen
After the desired parameters and return data have been specified you can enter a label and save the screen setup. (The screen may also be saved after it is run on the result grid view)

Screen Builder
Screen through the database of imported securities using screen parameters to restrict the securities returned and fields to return to restrict the data points returned for each security.
Symbol Screening
Add individual symbols to include in screen results. The added symbols will appear as a list above the entry field.

Company
Screen for a specific company name (Currently only accepts one company at a time)

DWA Sector
Screen by a particular Dorsey Wright Sector (Currently only accepts a single sector at a time)

Macro Sector
Default Behavior
The default behavior is to return all fields. Therefore, Select All and Clear Selection have the same initial effect. The usage of select all and clear selected is directed toward assisting in selection of many or few fields versus explicitly selecting all or none to return.

Select All
Selects all listed fields.

Clear Selected
Clears all selected fields.
